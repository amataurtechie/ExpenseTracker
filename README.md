Expense Tracker API 💰

A simple and scalable RESTful API built using Spring Boot to manage daily expenses.

**🚀 Features**
Add new expenses
View all expenses
Update expense details
Delete expenses
RESTful API design
**🛠️ Tech Stack**
Java
Spring Boot
Spring Data JPA
H2 / MySQL
Maven/Gradle
**📌 API Endpoints**
Method	Endpoint	Description
POST	/expenses	Add expense
GET	/expenses	Get all expenses
PUT	/expenses/{id}	Update expense
DELETE	/expenses/{id}	Delete expense
📥 Sample Request
{
  "title": "Food",
  "amount": 250,
  "date": "2026-04-18"
}
